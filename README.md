# Nationalize.io (nationalize)

Free REST API that predicts the nationality of a person based on their first name using probabilistic models derived from a dataset of approximately one billion people spanning 250 countries and territories. Accepts single or batched names, handles diacritics and non-Latin alphabets, and returns ranked country probabilities as JSON. The same API key works across all three Demografix services (Genderize, Agify, Nationalize).

## APIs

### Nationalize.io API

Predicts the nationality of a person based on their name, returning ranked country probabilities drawn from a dataset of approximately one billion people. Supports single name lookups, batched requests of up to 10 names, full name parsing, diacritics, and optional country scoping.

- **Human URL:** [https://nationalize.io/documentation](https://nationalize.io/documentation)
- **Base URL:** `https://api.nationalize.io`
